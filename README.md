🎥 **Project Demo:** https://drive.google.com/file/d/1iDs4vHgFwbwytsQGQdb71AQlpBBopzd_/view?usp=sharing

## 📌 Overview

The **AI Beauty Recommendation System** is a Machine Learning-based web application that provides personalized makeup recommendations based on a user's skin type, skin tone, face shape, occasion, makeup style, and budget.

The application predicts the most suitable makeup look and recommends beauty products from different price ranges through an interactive Streamlit interface.

---

## ✨ Features

* Personalized makeup recommendations
* AI-based makeup look prediction
* Budget-friendly, Mid-range, and Luxury product suggestions
* Interactive Streamlit web interface
* Modern and responsive UI
* Data visualization using Seaborn and Matplotlib

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/lakshmikantkulkarni096-dev/AI-Beauty-Recommendation-System.git
```

### Navigate to the project folder

```bash
cd AI-Beauty-Recommendation-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
